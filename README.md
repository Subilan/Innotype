# Innotype

![](https://i.loli.net/2019/02/02/5c55891c4fb3c.png)

Innotype 是一个**极其简单**的入门级静态博客程序，支持 Markdown 写作。Innotype 使用 [Google Material Design](//materia.io) 设计语言，前端框架为 [MDUI](//mdui.org)。

## 特点

Innotype 是一种静态博客，自然具有比动态博客更加便捷、迅速的特点。静态博客无法添加扩展性的功能，但如果您追求的仅仅是纯纯的写作的话，Innotype 也许是您不赖的选择。如果 Innotype 看起来不是那么适合您，您也可以去考虑考虑 @SomeBottle 开发的 [OBottle](//github.com/SomeBottle/OBottle)。

Innotype 具有如下特点：

- 静态，适合简单的写作、发布、分享、评论等。
- 采用前沿的 Material Design 设计。
- JQuery 编写 JS 代码，使程序维护和协助开发更加简便。
